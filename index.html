Github Tutorial

Step 1:

Creating a folder, named tutorial, on the local machine and initalizing it as git-repository

git init

Step 2:

Create index.html file in the tutorial folder. check status by using :

git status

Step 3:

The file is untracked. Add the file in the stagging area by :

git add index.html

Step 4:

Now commit the file by :

git commit -m "commit message"

Step 5 :

Now we are going to push the repository on gitHub

git remote add origin git@github.com:NimraHaq/gitHub.git git branch -M main git push -u origin main

The files in folder toturial are pushed to github repository.
